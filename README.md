# 🌐 Projeto Social

## 📌 Descrição do Projeto

O **Projeto Social** é uma página web desenvolvida para apresentar links e acessos rápidos para diferentes redes sociais. A interface simula uma tela de smartphone, permitindo que o usuário navegue entre perfis de redes sociais por meio de ícones interativos.

O objetivo do projeto é criar uma experiência visual simples e moderna, centralizando diferentes redes sociais em um único ambiente.

A página apresenta uma interface responsiva e organizada, facilitando a navegação e o acesso aos conteúdos.

---

## 🎯 Objetivo do Projeto

O objetivo deste projeto é praticar conceitos fundamentais do **desenvolvimento front-end**, especialmente relacionados à construção de interfaces e navegação entre conteúdos.

Durante o desenvolvimento foram trabalhados conceitos como:

- Estruturação semântica com HTML
- Estilização de layouts com CSS
- Uso de ícones e elementos visuais
- Organização de páginas web
- Estruturação de projetos front-end

---

## 🚀 Funcionalidades

O projeto possui as seguintes funcionalidades:

- Interface que simula a tela de um **smartphone**
- Ícones de **redes sociais interativos**
- Navegação entre diferentes páginas ou conteúdos
- Layout simples e organizado
- Estrutura visual moderna

---

## 🛠 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** → Estrutura da página
- **CSS3** → Estilização e layout da interface
- **Git e GitHub** → Controle de versão do projeto

Essas tecnologias são amplamente utilizadas no desenvolvimento de páginas web modernas.
